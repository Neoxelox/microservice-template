# microservice-template
**Complete Golang Microservice Template**

## Features
- [x] Golang
    - [x] Static compiling
    - [x] Dockerfile
    - [x] Echo Web Framework
    - [ ] GORM
    - [x] Panic-Recovery
    - [x] Logrus
- [ ] AutoTLS with Let's Encrypt
- [x] Docker-Compose that inits all services
- [x] Postgres
- [x] PgAdmin4
- [x] Prometheus
- [x] Jaeger
- [x] Grafana
- [ ] NewRelic
